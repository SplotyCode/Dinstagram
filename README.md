# Dinstagram
========================================================================
Dinstagram is a Instagram Clone that focusses on scaling.
It uses MondoDB and Redis as Datase. (Redis is only for caching stuff).
The Server part is programed in Java.
The Client is made in Html/Javascript.
========================================================================
